Sobre mim

Entrei na IBM pelo programa de estágio do Associates e trabalhei como estagiária e colaboradora efetivada em um projeto fazendo planos de testes, testando e desenvolvendo novas funções para o OSM (order system management) que é um sistema orquestrador que gerencia ordens de serviço, realizando comunicações com outros sistemas para entrada e processamento de Pedidos. Esse projeto usava a metodologia ágil Scrum e o desenvolvimento era realizado usando as tecnologias e ferramentas: XQuery, XPath, Xml, Java, JUnit, SpringBoot, Maven, Oracle Communications Design Studio/Eclipse e SoapUI.

Trabalhei também em outro projeto na IBM fazendo testes e desenvolvendo em Shell Script na ferramenta Multi Mediation da Ericson, que utiliza a linguagem proprietária DUP (Data Unit Processing). Nesse projeto ajudei no desenvolvimento de uma API em Shell Script com o objetivo de acessar a ferramenta Ericsson Multi Mediation usando a interface de comando (CLI) com os comandos específicos da ferramenta.

Trabalhei como voluntária na In Junior, empresa júnior do Instituto de Computação da Universidade Federal Fluminense, como Desenvolvedora Backend ajudando no desenvolvimento inicial de um projeto interno que utilizava Ruby on Rails e Assessora de Vendas.
